Detailed Description

Defines text repeat elements with font styling and justification.

Text repeat elements are indications like "D.S. al Fine", "Segno", or "To Coda" that have playback jumps associated with them.

The cmper is the unique identifier used in the document.

This class is identified by the XML node name "textRepeatDef".
