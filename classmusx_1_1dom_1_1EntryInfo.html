Detailed Description

Information an entry along with the entry.

This class is used in iteration functions to supply information about the entry along with the entry itself.

Its pointers are owned by EntryFrame.

◆ cumulativeRatio

util::Fraction musx::dom::EntryInfo::cumulativeRatio {}

the cumulative tuplet ratio in effect at the time of this entry. This value allows a tuplet to discover the ratio of all the other active tuplets by backing out its own ratio.
