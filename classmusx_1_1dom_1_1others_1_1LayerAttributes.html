Detailed Description

Represents the attributes of a Finale "layer".

The cmper is the layer ID (0-3). In the Finale UI these are represented as Layer 1 thru Layer 4. The 0-based cmper value is used to identify layers where needed, rather than the 1-based UI value.

This class is identified by the XML node name "layerAtts".
