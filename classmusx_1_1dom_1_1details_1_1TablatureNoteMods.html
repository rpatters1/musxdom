Detailed Description

Specifies the TAB string a note appears on. Finale automatically figures out the fret number from the open string note and fret intervals specified in the staff's associated others::FretInstrument.

Entry::noteDetail is set if any note in the entry has tablature note mods.

This class is identified by the XML node name "tabAlter".
