Detailed Description

Per-instance positioning overrides for TextRepeatDef.

The cmper is the cmper for the corresponding TextRepeatDef. The inci allows multiple instances for this definition throughout the document. Use RepeatIndividualPositioning::measureId to find the instance for a specific measure.
