Detailed Description

Represents a graphic assignment anchored to a specific staff and measure.

Cmper1 is the staff (staffId) Cmper and Cmper2 is the measure Cmper.

Positions and sizes are Finale layout units: horizontal values use Evpu from the measureâ€™s left edge (system coordinates); vertical values use Evpu from the staff baseline.

This class is identified by the XML node name "measGraphicAssign".
