Detailed Description

Represents a text repeat assignment with positioning and behavior properties.

The cmper is the cmper of the Measure that has this item.

This class is identified by the XML node name "textRepeatAssign".

Member Function Documentation

◆ calcTargetMeasure()

std::optional< MeasCmper > musx::dom::others::TextRepeatAssign::calcTargetMeasure ( ) const

Calculates the target measure for this text repeat assignment, if any.

Returns the explicit or relative target when the action specifies a concrete measure. For JumpToMark, it resolves the target repeat ID to the referenced assignment. Returns std::nullopt for actions that do not provide a measure target.

Returns
The target measure for the jump, or std::nullopt if none applies.
