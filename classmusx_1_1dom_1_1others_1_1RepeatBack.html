Detailed Description

Represents a repeat-backward marker with positioning and behavior properties.

The cmper is the cmper of the Measure that has this item.

This class is identified by the XML node name "repeatBack".

int musx::dom::others::RepeatBack::passNumber {}

Specifies the pass number for the trigger and depends on the value of trigger. (xml tag is <actuate>) If trigger is UntilPass, this specifies the number of times to play the section. If trigger is OnPass, this specifies the pass on which to jump.
