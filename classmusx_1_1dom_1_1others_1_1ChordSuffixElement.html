Detailed Description

Represents a single element in a chord suffix (e.g., a symbol like "1" or "+").

Each chord suffix consists of one or more such elements, each with its own font and positioning. The inci is a 0-based index for the element.

The cmper is a unique identifier used by details::ChordAssign to reference the suffix and its playback information. (See ChordSuffixPlayback.) Suffixes are a library from which chord assignments select their particular suffix.

This class is identified by the XML node name "chordSuffix".
